# Neuronforge -- Model Training & Fine-Tuning Toolkit

**Train custom models, manage weights, experiment tracking, and hyperparameter optimization. Built for researchers who demand full control over the training process.**

## Quick Start

```bash
pip install neuronforge
```

## Features

- First-class AMD ROCm support
- Production-ready out of the box
- Modular and extensible design
- Comprehensive documentation
